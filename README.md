### NLP Amazon review sentimental prediction  Processed >100k public Amazon review text data to predict binary sentiment of 'positive' or 'negative' #### Description  1. proprocess text (tokenize, lower case, lemmatize, remove stopwords)2. trained model with NLTK, sklearn TF-IDF, MultinomialNB, accuracy 0.88  3. saved optimized model to .pkl  ![img](img/work_flow.png)#### Data[Amazon product reviews, Kindel Store 5-score data](http://jmcauley.ucsd.edu/data/amazon/)  This dataset contains product reviews and metadata from Amazon, including 142.8 million reviews spanning May 1996 - July 2014.This dataset includes reviews (ratings, text, helpfulness votes), product metadata (descriptions, category information, price, brand, and image features), and links (also viewed/also bought graphs).#### TODOuse Doc2vec + keras.LSTM  interactive flask app to predict new review with saved model
